# Kushi Cabs Website - Google Play Store Compliant

## Project Overview

This is a complete, professional website for Kushi Cabs - a comprehensive mobility platform for drivers, vendors, and administrators. The website is fully compliant with **Google Play Store requirements** for app privacy policies and data handling.
